	# Documentação do Software de Gerenciamento de Estoque

    ## Introdução
    Bem-vindos ao guia do nosso software de gerenciamento de estoque. Este documento fornece instruções detalhadas sobre como navegar e utilizar as funcionalidades do aplicativo.

    ## Cadastro e Login

    ### Primeiro Acesso e Cadastro
    - Ao iniciar o software pela **primeira vez**, o usuário será recebido pela tela de **login**.
    - Para novos usuários, é necessário **criar uma conta** clicando no link de cadastro.
    - No formulário de cadastro, insira seu **nome**, **email** e **senha**.
    - Após a criação da conta, você será redirecionado para a **página de login**.
    - Entre com seu **email** e **senha** cadastrados para acessar o aplicativo.

    ## Página de Registros

    ### Navegação e Pesquisa
    - Na página de registros, você pode **pesquisar** itens pelo **nome**, **descrição**, **ID** ou **patrimônio**.
    - É possível visualizar o **estoque atual**, **registros de entrada** e **registros de saída**.

    ### Registro de Entrada
    - Para registrar a entrada de um item, vá até a aba **Registro de Entrada**.
    - Selecione no **checkbox** se o produto **tem** ou **não tem patrimônio**.
    - Se **tiver patrimônio**, a quantidade será **1** por padrão, pois cada patrimônio é único.
    - Se **não tiver patrimônio**, você deverá inserir a **quantidade** do item.

    ### Registro de Saída
    - Para dar saída em um produto, digite o **patrimônio** ou o **ID** do produto.
    - Se o produto for encontrado pelo **patrimônio**, a quantidade será definida como **1** automaticamente.
    - Se o produto for encontrado pelo **ID**, a **quantidade** poderá ser editada, respeitando o **máximo em estoque**.

    Este guia foi criado para facilitar o uso do software e garantir uma gestão eficiente do seu estoque.
